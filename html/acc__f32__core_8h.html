Detailed Description

AccF32 component API.

Lifecycle: create -> (step / steps / reset)* -> destroy

Example:

acc_f32_state_t *obj = acc_f32_create(0.0f);
acc_f32_step(obj, 0.0f);
acc_f32_destroy(obj);
