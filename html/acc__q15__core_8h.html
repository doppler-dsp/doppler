Detailed Description

AccQ15 component API.

Lifecycle: create -> [step / steps / reset]* -> destroy

Example:

acc_q15_state_t *obj = acc_q15_create(0);
acc_q15_step(obj, 0);
acc_q15_destroy(obj);
