Detailed Description

Local oscillator: NCO + 2^16 sin/cos LUT → CF32 phasors.

Re-implemented from the doppler reference (native/src/lo/lo_core.c). Algorithm and AVX-512 paths are identical; the state struct and API use doppler conventions (double norm_freq, jm property names).

Generate n CF32 phasors at the current norm_freq.

Output is emitted before increment: out(0) corresponds to the phase at entry, out(1) to phase + phase_inc, etc. Returns n.

◆ lo_steps_ctrl()

size_t lo_steps_ctrl ( lo_state_t * state,
const float * ctrl,
size_t ctrl_len,
float complex * out )

Generate CF32 phasors with per-sample FM deviation.

ctrl(i) (real float, fractional part used) is converted to a per-sample phase-increment delta added on top of the base phase_inc. The base norm_freq is not modified.

Output length equals ctrl_len. Returns ctrl_len.
