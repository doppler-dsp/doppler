Detailed Description

Delay component API.

Lifecycle: create -> (step / steps / reset)* -> destroy

Example:

delay_state_t *obj = delay_create();
float complex y = delay_step(obj, 0.0f + 0.0f * I);
delay_destroy(obj);

◆ delay_create()

delay_state_t * delay_create ( size_t num_taps)

Create a delay instance.

Allocates a dual-buffer circular delay line of length num_taps. The internal buffer is rounded up to the next power of two for efficient modular addressing.

Parameters
num_tapsWindow length (>= 1). Rounded up internally.
Returns
Heap-allocated state, or NULL on allocation failure.
Note
Caller must call delay_destroy() when done.
