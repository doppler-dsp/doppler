Resampler_state_t * Resampler_create ( double rate)

Create a Resampler with the built-in 4096×19 Kaiser bank.

Parameters
rateResample ratio (out/in). Values >= 1.0 interpolate; values < 1.0 decimate.
Returns
Non-NULL on success, NULL on OOM.
