Detailed Description

Real-to-complex halfband 2:1 decimator (Architecture D2).

Lifted from c/src/hbdecim.c (dp_hbdecim_r2cf32_t section).

Algorithm: Input is split into even (x[2m]) and odd (x[2m+1]) streams, each stored in its own dual-write circular delay line. One output per pair:

N even (fir_on_even=1): I = antisymmetric FIR on even stream Q = delay_sign × 0.5 × odd[centre]

N odd (fir_on_even=0): Q = antisymmetric FIR on odd stream (at offset +1) I = delay_sign × 0.5 × even[centre]

Output: y[m] = (I + jQ) × (-1)^m (parity flip provides fs/4 shift)

Coefficients are baked as h_mod[k] = h_fir[k] × (-1)^k × 0.5.
