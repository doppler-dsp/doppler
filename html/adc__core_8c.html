adc_state_t * adc_create ( int bits,
float dbfs,
int dithering )

Create an ADC instance.

Parameters
bitsADC resolution in bits (1..64).
dbfsFull-scale reference level in dBFS (typically negative). A -10 dBFS signal at amplitude 10^(dbfs/20) fills the ADC range.
dithering0 = no dither; non-zero = TPDF dither enabled.
Returns
Heap-allocated state, or NULL on invalid args or allocation failure.
Note
Caller must call adc_destroy() when done.

JM_HOT void adc_steps ( adc_state_t * state,
const float * input,
int64_t * output,
size_t n )

Process a block of samples.

When dithering is disabled the float-to-scale multiply is widened with SIMD (jm_simd.h); the int64_t conversion and clamp remain scalar. When dithering is enabled the loop is scalar to preserve sequential PRNG state.

Parameters
stateComponent state (mutated; clipped flag updated).
inputInput array (length >= n).
outputOutput array (length >= n).
nNumber of samples.
