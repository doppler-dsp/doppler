typedef struct dp_msg dp_msg_t

Opaque zero-copy message handle returned by recv functions.

The data buffer is valid until dp_msg_free() is called. Use the accessor functions to retrieve a pointer to the sample data, size, etc.

◆ dp_sample_type_t

enum dp_sample_type_t

Selects the wire format of complex samples.

All sample arrays are packed as interleaved I/Q pairs.

Values are fixed — new types are appended to preserve wire compatibility with older receivers.

Enumerator
CI32 

Complex int32: int32_t I/Q (8 bytes/sample).

CF64 

Complex float64: double I/Q (16 bytes/sample).

CF128 

Complex long double I/Q (32 bytes/sample).

CI8 

Complex int8: int8_t I/Q (2 bytes/sample).

CI16 

Complex int16: int16_t I/Q (4 bytes/sample).

CF32 

Complex float32: float I/Q (8 bytes/sample).
