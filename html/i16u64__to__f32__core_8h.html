Detailed Description

I16U64ToF32 component API.

Lifecycle: create -> (step / steps / reset)* -> destroy

Example:

i16u64_to_f32_state_t *obj = i16u64_to_f32_create(32768.0f);
float y = i16u64_to_f32_step(obj, 0U);
i16u64_to_f32_destroy(obj);
