Detailed Description

Spectral module — public C API.

Provides windowing (Kaiser, Hann), ENBW computation, magnitude conversion, and peak finding. These are pure functions with no persistent state.

size_t find_peaks_f32 ( const float * db,
size_t n,
size_t n_peaks,
float min_db,
dp_peak_t * out )

Find up to n_peaks local maxima in a DC-centred F32 dB spectrum.

Algorithm:

  1. Local-max scan: db(k) > db(k-1) && db(k) >= db(k+1), above min_db.
  2. Parabolic interpolation for sub-bin frequency accuracy.
  3. Sort descending by amplitude; return top n_peaks.
Parameters
dbF32 dB spectrum, DC-centred, length n. Must be >= 3.
nNumber of bins.
n_peaksMaximum number of peaks to return.
min_dbAmplitude threshold; bins below this are ignored.
outCaller-allocated output array, length >= n_peaks.
Returns
Number of peaks written (<= n_peaks).
