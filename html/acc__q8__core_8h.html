Detailed Description

AccQ8 component API.

Lifecycle: create -> [step / steps / reset]* -> destroy

Example:

acc_q8_state_t *obj = acc_q8_create(0);
acc_q8_step(obj, 0);
acc_q8_destroy(obj);
