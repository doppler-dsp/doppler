Open a ZMQ PUB sink.

Parameters
endpointZMQ bind endpoint, e.g. "tcp://0.0.0.0:5555".
sample_typeWire type (wavegen order): 0 cf32, 1 cf64, 2 ci32, 3 ci16, 4 ci8. Integer types use full-scale ±1.0.
Returns
