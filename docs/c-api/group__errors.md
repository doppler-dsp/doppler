## Detailed Description


Every send/recv function returns one of the shared doppler error codes (DP\_OK, DP\_ERR\_INIT, DP\_ERR\_SEND, DP\_ERR\_RECV, DP\_ERR\_INVALID, DP\_ERR\_TIMEOUT, DP\_ERR\_MEMORY) defined in [**clib\_common.h**](clib__common_8h.md), included above. Use [**dp\_strerror()**](group__utils.md#function-dp_strerror) to obtain a human-readable description.
