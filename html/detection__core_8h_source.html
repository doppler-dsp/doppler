142/* ── Power detector ──────────────────────────────────────────────────────── */
143/* */
144/* The power detector uses power_stat = |R[0]|² / mean(|R[τ]|²) instead of */
145/* the envelope ratio. Under H0 (noise only): */
146/* */
147/* power_stat ~ Exponential(1) → P(power_stat > p) = exp(-p) */
148/* */
149/* The threshold is simply p = -ln(Pfa), and the detection probability is */
150/* */
151/* Pd = Q_1(sqrt(2·dwell·snr_power), sqrt(2·p)) */
152/* */
153/* where snr_power = snr_amplitude^2 (signal power / noise power per sample */
154/* at the correlator output). */
155/* */
156/* Since sqrt(2·p) = sqrt(-2·ln(Pfa)) = det_threshold(Pfa), the Pd formula */
157/* is identical to the envelope case expressed in power units. The two */
158/* detectors are equivalent in performance (same Pd at the same SNR in dB); */
159/* the power detector offers a simpler threshold formula and an exponential */
160/* null distribution. */
